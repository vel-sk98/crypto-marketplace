# 🚀 Crypto Dashboard (React)

A modern and responsive **Crypto Dashboard** built using React.  
This application displays real-time cryptocurrency market data with a clean UI and optimized performance.

## 🛠️ Tech Stack

- React  
- JavaScript  
- ContextAPI  
- React Router DOM  
- Chart Library (Recharts)  
- CSS  

## ✨ Features

- 📈 Live cryptocurrency price tracking  
- 🔍 Search and filter functionality  
- 📊 Interactive price charts  
- 💹 Market overview and trends  
- 📱 Responsive design  

## 🚧 Project Status

This project is currently **under active development**.  
I am continuously working on improving the UI, optimizing performance, and adding new features.
